AI健康カルテ Ver.14.1 完成版

修正内容：
- 数字や文字の入力中は、下部の固定ボタンを自動で非表示
  ・集計
  ・コピー
  ・この日を削除
- 入力中は「上へ」ボタンも非表示
- キーボードを閉じると固定ボタンが自動で再表示
- 体重の整数欄に2桁入力すると、小数欄へ自動移動
- 排尿前体重・排尿後体重も同じ動作
- 100kg以上を入力する場合は、続けて3桁目を入力可能
- Ver.14までの飲み物選択、比較表、血糖値2単位、カードジャンプ等を維持

例：
96.20kg → 左に96と入力すると、右の小数欄へ自動移動
98.70kg → 左に98と入力すると、右の小数欄へ自動移動
